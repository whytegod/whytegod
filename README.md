# WHYTGOD

**Whytegod** is an AI-first, peer-to-peer financial execution layer built on top of the **Open Financial Network (OFN)**.

It serves as the coordination, intelligence, and routing system for decentralized finance operations across borders, assets, and counterparties.

---

## Architecture Overview

WHYTGOD is part of a three-layer system:

### 1. Open Financial Network (OFN)
Core financial protocol and infrastructure.
- Ledger logic
- Security primitives
- Settlement rules

Repository:
https://github.com/whytegod/open-financial-network

---

### 2. WHYTGOD (This Repository)
Execution and intelligence layer.
- AI-assisted P2P routing
- Risk-aware trade coordination
- Agent-based financial execution
- Human + machine collaboration layer

This repo will contain:
- Execution logic
- Agent definitions
- Coordination modules
- API interfaces

---

### 3. WHYTGOD Website
Public-facing manifesto and documentation.

Live at:
https://whytegod.github.io

---

## Vision

WHYTGOD is designed to:
- Remove friction from peer-to-peer finance
- Enable trust-minimized human exchange
- Bridge informal finance with institutional-grade logic
- Scale intelligence, not bureaucracy
