Whytegod

Whytegod is an AI-first, peer-to-peer financial execution protocol specification built on top of the Open Financial Network (OFN).

It defines a standardized, intent-based execution layer for financial actions, where users express what they want to achieve, and compliant systems determine how execution occurs.

Whytegod is not an application.
It is not an exchange.
It is a protocol specification.

Purpose

Modern financial systems force users to manage execution details manually. Whytegod removes this burden by separating intent from execution complexity.

  • Routing decisions
  • Counterparty discovery
  • Timing and settlement logic
  • Trust assumptions

The protocol enables systems where humans define goals, and machines coordinate execution without custodial control.

Design Principles

  • Decentralization by default
  • Non-custodial execution
  • Protocol over platform
  • Simple enough for the ordinary man
  • Composable and permissionless
